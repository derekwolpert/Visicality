Welcome to vi路si路cal路i路ty (visual + musicality)


vi路si路cal路i路ty is a web-based music visualizer utilizing Web Audio API and D3.js.


Note: For best compatibility please use a desktop version of Google Chrome. With other web browsers some features may not be available. While this project's UI / UX is optimized for desktop browsers, it is also compatible with mobile devices that support Web Audio API.


Features:


  • Visualize audio files stored locally on your own device.
  • Seamlessly switch between nine visualizer designs.
  • Select from nine diverse visualization color schemes.
  • Full costomization for background color gradient. Select any three colors for the background gradient, set the gradient's direction. The page's favicon will update accordingly.
  • Adjust the gain of the audio, and watch as the intensity of the visualizer changes.
  • Quick access to transition in and out of full-screen mode.
  • Intuitive UI / UX that automatically hides when 1) audio is playing, and 2) there is no detected mouse movement.
  • Flexible combination of onscreen controls and keyboard shortcuts.

Keyboard Controls

In addtion to on-screen controls, vi路si路cal路i路ty supports a wide range of keyboard shortcuts:

Key Action
Spacebar Play / Pause Audio
Left-Arrow Seek audio 5 seconds backwards
Right-Arrow Seek audio 5 seconds forward
Up-Arrow Increase gain (volume) by 10 percent
Down-arrow Decrease gain (volume) by 10 percent
F Enter / Exit fullscreen mode
W Previous visualizer design, based on associated number
S Next visualizer design, based on associated number
A Previous visualizer color, based on associated number
D Next visualizer color, based on associated number
R Set random three-color background gradient
Q Rotate background left 45 degrees
E Rotate background right 45 degrees
